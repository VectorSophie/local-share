## 로컬 파일 공유 앱

왜 만들었냐.

카카오톡이랑 구글 드라이브 옮겨다니는거 귀찮아서 만들었다.

안정적이게 로컬 리눅스 머신에서 띄울거다.

꼭! 서버 머신과 같은 wi-fi에 연결되어있어야함...

```pipx install flask```

로 설치(리눅스라서 pipx씀)

```pipx run --spec flask python server.py```

로 시작해서 서버 머신 주소:5000(예: https://127.0.0.1:5000 )로 접속하면 됨...

현재는 UI가 매우, 매우 최소임.

로컬 머신 /uploads 폴더로 파일을 볼수 있다.
